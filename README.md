# [New Specimen REDUX](https://gamebanana.com/mods/641169) - A(nother) Funkin' Mod

![](/art/freeplayIcons.gif)

## Installation

Just click the green `<> Code` button up at the top and then click `Download ZIP` in the dropdown menu.

Once the file finishes installing, extract the zip to the `mods` folder inside of your [Friday Night Funkin'](https://github.com/FunkinCrew/Funkin/) installation.

*(This modpack is only compatable with API [v0.7.0](https://github.com/FunkinCrew/Funkin/releases/tag/v0.7.2) and [newer](https://github.com/FunkinCrew/Funkin/releases))*

---

Your installation folder should look something like this:

```
-assets
-manifest
-plugins
-backup
-logs
-screenshots
-mods
 |-NewSpecimenREDUX_modpack-public-main        [<--- This is where you extracted the zip!]
-Funkin.exe
```

Feel free to remove the `art` folder along with the `.gitattributes` and `README.md` files inside the modpack once you've installed it correctly.
