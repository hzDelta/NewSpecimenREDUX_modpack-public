# Changelog

All notable changes will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-02-22

### Added

- [MOBILE] The official mobile port is out! All neccesary assets have been converted to the ASTC format to boost performance on older devices.

- Added the 'Legacy' 2023 versions of New Specimen and Hijacked.
  - These were only Alternate Instrumentals for said songs, but now they're fully playable!
    - The REDUX versions of these tracks will be available as Alternate Instrumentals in Freeplay.

- Added a new stage: 'Admin [Legacy]'

### Removed

- Removed the `PanicHUD_Lite` module.
  - The module has since been updated and is now part of the *PolusUtils* utility mod which should release alongside this update.

## [1.0.1] - 2026-01-04

### Changed

- Bumped the API version up to `0.8.0`.
- Made a slight change to the `PanicHUD_Lite` module.

## [1.0.0] - 2025-12-25

### Added

- Added a Story Mode level 'SKELD SKIRMISH' with two new songs: New Speciemn and Hijacked.
  - These songs have two Alternate Instrumentals accessable in Freeplay.
  - The new stage 'Navigation Hall' has unique Pause and Game Over tracks alongside a custom HUD during gameplay.

- Added a new variant of Gettin' Freaky that plays in the menus.
